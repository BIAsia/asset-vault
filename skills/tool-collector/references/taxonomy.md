# Taxonomy Guidance

Use stable, broad tags. Prefer adding discovery words as `aliases` instead of creating many narrow tags.

Good tag examples: `graphics`, `shader`, `animation`, `typography`, `vector`, `stroke`, `rendering`, `creative-coding`, `library`, `design`, `interaction`.

Aliases should include Chinese search words and common English variants. For example:

- `stroke`: 线条, 描边, 笔触, 文本描边
- `shader`: 着色器, 特效, GPU 效果
- `typography`: 文字, 字体, 排版, 文本
